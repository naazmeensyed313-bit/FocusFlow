# FocusFlow – Virtual Study Room

FocusFlow is a desktop productivity application built using Python and CustomTkinter.  
The application provides a modern study environment with productivity tools such as task management, study timer, notes management, and streak tracking through a clean multi-page dashboard interface.

---

## Features

- Modern dashboard-based UI
- Multi-page navigation system
- Study timer with session controls
- Notes management system
- Task management with completion tracking
- Study streak monitoring
- Persistent local storage using JSON
- Dark themed responsive interface
